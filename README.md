**Business Problem**

Mobile app companies need to understand how users interact with their devices in order to improve engagement, optimize app performance, and target high-value user segments.

This project analyzes a dataset of 700 mobile users to uncover patterns in:

- device usage
- application engagement
- screen time
- data consumption
- user activity levels

The objective is to identify high-engagement user segments and device usage trends that could inform product and marketing decisions.


**Dataset Description**

The dataset contains behavioral data from 700 mobile device users, including:

- Device model
- Operating system
- App usage time
- Screen-on time
- Number of installed apps
- Data usage
- Battery consumption
- Age group
- Gender
- User behavior classification
This dataset enables analysis of mobile engagement patterns across devices and demographics.


**Tools Used**
- Excel
- Data Cleaning – preparing dataset for analysis
  

**Key Questions Answered**

- Which device models have the highest user engagement?
- How does screen-on time differ across device types?
- What percentage of users are heavy, moderate, or light users?
- Do heavy users install more applications than light users?
- How does operating system influence data consumption?
- Which age groups show the highest mobile engagement?

**Key Insights**

- Heavy users represent the largest segment (40%), indicating strong mobile engagement across the user base.
- Android dominates the dataset (79%), but iOS users consume nearly equal data, suggesting high engagement per iOS user.
- Heavy users install significantly more apps (~79) compared to light users (~23), indicating that application variety may drive engagement.
- Screen-on time averages around 5 hours daily, suggesting consistent device usage across most devices.
- Younger age groups (18–34) show higher proportions of heavy usage patterns.


**Business Recommendations**

Based on the analysis:

- Product teams should prioritize features that support high-engagement users, who represent the largest segment.
- App developers should optimize performance for popular devices like Xiaomi Mi 11 and iPhone 12.
- Marketing teams could target younger demographics, who demonstrate higher device engagement.
- Developers should consider battery optimization strategies, as higher engagement correlates with higher battery consumption.
